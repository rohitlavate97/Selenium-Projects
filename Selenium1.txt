## ğŸ“„ Step 3: Create Page Objects (1-2 hours)

### 3.1 Create Your First Page Object

Let's create a page object for your Login Page.

**File:** `src/main/java/com/automation/pages/LoginPage.java`

```java
package com.automation.pages;

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.support.FindBy;
import com.automation.base.BasePage;

/**
 * Login Page Object
 * Represents the login page of YOUR_WEBSITE_NAME
 */
public class LoginPage extends BasePage {
    
    // ============================================
    // PAGE ELEMENTS
    // ============================================
    // Replace these locators with YOUR actual elements
    
    @FindBy(id = "username")
    private WebElement usernameField;
    
    @FindBy(id = "password")
    private WebElement passwordField;
    
    @FindBy(id = "loginBtn")
    private WebElement loginButton;
    
    @FindBy(id = "rememberMe")
    private WebElement rememberMeCheckbox;
    
    @FindBy(linkText = "Forgot Password?")
    private WebElement forgotPasswordLink;
    
    @FindBy(css = ".error-msg")
    private WebElement errorMessage;
    
    @FindBy(css = ".signup-link")
    private WebElement signUpLink;
    
    // ============================================
    // CONSTRUCTOR
    // ============================================
    
    public LoginPage(WebDriver driver) {
        super(driver);
    }
    
    // ============================================
    // PAGE ACTIONS
    // ============================================
    
    /**
     * Enter username
     * @param username Username to enter
     * @return LoginPage for method chaining
     */
    public LoginPage enterUsername(String username) {
        logger.info("Entering username: {}", username);
        waitForVisible(usernameField);
        sendKeys(usernameField, username);
        return this;
    }
    
    /**
     * Enter password
     * @param password Password to enter
     * @return LoginPage for method chaining
     */
    public LoginPage enterPassword(String password) {
        logger.info("Entering password");
        waitForVisible(passwordField);
        sendKeys(passwordField, password);
        return this;
    }
    
    /**
     * Click login button
     */
    public void clickLoginButton() {
        logger.info("Clicking login button");
        waitForClickable(loginButton);
        click(loginButton);
        waitForPageLoad();
    }
    
    /**
     * Perform complete login action
     * @param username Username
     * @param password Password
     */
    public void login(String username, String password) {
        enterUsername(username);
        enterPassword(password);
        clickLoginButton();
    }
    
    /**
     * Check/uncheck remember me
     * @param select true to check, false to uncheck
     */
    public LoginPage selectRememberMe(boolean select) {
        if (select && !rememberMeCheckbox.isSelected()) {
            click(rememberMeCheckbox);
        } else if (!select && rememberMeCheckbox.isSelected()) {
            click(rememberMeCheckbox);
        }
        return this;
    }
    
    /**
     * Click forgot password link
     */
    public void clickForgotPassword() {
        logger.info("Clicking forgot password link");
        click(forgotPasswordLink);
        waitForPageLoad();
    }
    
    /**
     * Click sign up link
     */
    public void clickSignUp() {
        logger.info("Clicking sign up link");
        click(signUpLink);
        waitForPageLoad();
    }
    
    // ============================================
    // PAGE VERIFICATIONS
    // ============================================
    
    /**
     * Get error message text
     * @return Error message
     */
    public String getErrorMessage() {
        waitForVisible(errorMessage);
        return getText(errorMessage);
    }
    
    /**
     * Check if error message is displayed
     * @return true if error displayed
     */
    public boolean isErrorDisplayed() {
        return isDisplayed(errorMessage);
    }
    
    /**
     * Verify all elements are present
     * @return true if all elements present
     */
    public boolean verifyAllElements() {
        logger.info("Verifying all login page elements");
        return isDisplayed(usernameField) &&
               isDisplayed(passwordField) &&
               isDisplayed(loginButton) &&
               isDisplayed(rememberMeCheckbox) &&
               isDisplayed(forgotPasswordLink);
    }
    
    /**
     * Check if login page is loaded
     * @return true if page loaded
     */
    @Override
    public boolean isPageLoaded() {
        try {
            waitForVisible(usernameField);
            waitForVisible(passwordField);
            waitForVisible(loginButton);
            logger.info("Login page loaded successfully");
            return true;
        } catch (Exception e) {
            logger.error("Login page did not load: {}", e.getMessage());
            return false;
        }
    }
}
```

## ğŸ”„ Step 4: Create Flow Classes (30 minutes - Optional but Recommended)

Flows contain business logic and multi-page interactions.

**When to Create Flows:**
- âœ… When action spans multiple pages
- âœ… When business logic is complex
- âœ… When you want to reuse logic across tests

**File:** `src/main/java/com/automation/flows/LoginFlow.java`

```java
package com.automation.flows;

import org.openqa.selenium.WebDriver;
import com.automation.base.BaseFlow;
import com.automation.config.ConfigReader;
import com.automation.pages.LoginPage;
import com.automation.pages.HomePage;

/**
 * Login Flow
 * Handles login-related business logic
 */
public class LoginFlow extends BaseFlow {
    
    public LoginFlow(WebDriver driver) {
        super(driver);
    }
    
    /**
     * Navigate to login page
     */
    public LoginPage navigateToLoginPage() {
        logInfo("Navigating to login page");
        String appUrl = ConfigReader.getApplicationUrl();
        driver.get(appUrl);
        waitForPageLoad();
        
        LoginPage loginPage = new LoginPage(driver);
        if (!loginPage.isPageLoaded()) {
            throw new RuntimeException("Login page did not load");
        }
        return loginPage;
    }
    
    /**
     * Perform login with default credentials
     */
    public HomePage performLogin() {
        String username = ConfigReader.getUsername();
        String password = ConfigReader.getPassword();
        return performLogin(username, password);
    }
    
    /**
     * Perform login with specific credentials
     */
    public HomePage performLogin(String username, String password) {
        logInfo("Performing login for: " + username);
        
        try {
            // Navigate and verify login page
            LoginPage loginPage = navigateToLoginPage();
            
            // Perform login
            loginPage.login(username, password);
            
            // Verify home page loaded
            HomePage homePage = new HomePage(driver);
            if (!homePage.isPageLoaded()) {
                throw new RuntimeException("Home page did not load after login");
            }
            
            logInfo("Login successful");
            return homePage;
            
        } catch (Exception e) {
            logError("Login failed: " + e.getMessage());
            throw e;
        }
    }
    
    /**
     * Attempt invalid login and return error message
     */
    public String performInvalidLogin(String username, String password) {
        logInfo("Attempting invalid login");
        
        LoginPage loginPage = navigateToLoginPage();
        loginPage.login(username, password);
        waitForPageLoad();
        
        if (loginPage.isErrorDisplayed()) {
            String errorMsg = loginPage.getErrorMessage();
            logInfo("Error message: " + errorMsg);
            return errorMsg;
        }
        
        return "";
    }
    
    /**
     * Perform logout
     */
    public void performLogout() {
        logInfo("Performing logout");
        HomePage homePage = new HomePage(driver);
        homePage.clickLogout();
        waitForPageLoad();
    }
}
```

package com.automation.flows;

import org.openqa.selenium.WebDriver;
import com.automation.base.BaseFlow;
import com.automation.pages.*;

/**
 * Purchase Flow
 * Handles complete purchase journey
 */
public class PurchaseFlow extends BaseFlow {
    
    public PurchaseFlow(WebDriver driver) {
        super(driver);
    }
    
    /**
     * Complete purchase flow
     */
    public String completePurchase(String productName, String quantity) {
        logInfo("Starting purchase flow for: " + productName);
        
        // Search for product
        HomePage homePage = new HomePage(driver);
        homePage.searchProduct(productName);
        
        // Select product
        ProductListPage listPage = new ProductListPage(driver);
        listPage.selectProduct(productName);
        
        // Add to cart
        ProductDetailPage detailPage = new ProductDetailPage(driver);
        detailPage.selectQuantity(quantity);
        detailPage.addToCart();
        
        // Go to cart
        CartPage cartPage = new CartPage(driver);
        cartPage.proceedToCheckout();
        
        // Complete checkout
        CheckoutPage checkoutPage = new CheckoutPage(driver);
        checkoutPage.enterShippingDetails();
        checkoutPage.selectPaymentMethod("Credit Card");
        checkoutPage.placeOrder();
        
        // Get order number
        OrderConfirmationPage confirmationPage = new OrderConfirmationPage(driver);
        String orderNumber = confirmationPage.getOrderNumber();
        
        logInfo("Purchase completed. Order number: " + orderNumber);
        return orderNumber;
    }
}

package com.automation.tests;

import org.testng.Assert;
import org.testng.annotations.*;
import com.automation.base.ExtentReportManager;
import com.automation.config.ConfigReader;
import com.automation.flows.LoginFlow;
import com.automation.pages.HomePage;
import com.automation.pages.LoginPage;

/**
 * Login Test Suite
 * Tests all login functionality
 */
public class LoginTest extends BaseTest {
    
    private LoginFlow loginFlow;
    
    @BeforeMethod(dependsOnMethods = "setUp")
    public void setupTest() {
        loginFlow = new LoginFlow(driver);
    }
    
    /**
     * Test: Successful login with valid credentials
     */
    @Test(priority = 1,
          groups = {"smoke", "regression", "login"},
          description = "Verify user can login successfully with valid credentials")
    public void testValidLogin() {
        // Test metadata
        ExtentReportManager.assignCategory("Smoke", "Login");
        ExtentReportManager.assignAuthor("QA Team");
        ExtentReportManager.logInfo("Test: Valid Login");
        
        // Arrange
        String username = ConfigReader.getUsername();
        String password = ConfigReader.getPassword();
        ExtentReportManager.logInfo("Using username: " + username);
        
        // Act
        HomePage homePage = loginFlow.performLogin(username, password);
        ExtentReportManager.logInfo("Login action completed");
        
        // Assert
        Assert.assertNotNull(homePage, "Home page should not be null");
        Assert.assertTrue(homePage.isPageLoaded(), "Home page should be loaded");
        Assert.assertTrue(homePage.isUserLoggedIn(), "User should be logged in");
        
        ExtentReportManager.logPass("âœ“ Valid login test passed");
    }
    
    /**
     * Test: Login fails with invalid password
     */
    @Test(priority = 2,
          groups = {"regression", "login", "negative"},
          description = "Verify error message for invalid password")
    public void testInvalidPassword() {
        // Test metadata
        ExtentReportManager.assignCategory("Regression", "Login", "Negative");
        ExtentReportManager.logInfo("Test: Invalid Password");
        
        // Arrange
        String validUsername = ConfigReader.getUsername();
        String invalidPassword = "WrongPassword123!";
        ExtentReportManager.logInfo("Testing invalid password scenario");
        
        // Act
        String errorMsg = loginFlow.performInvalidLogin(validUsername, invalidPassword);
        
        // Assert
        Assert.assertNotNull(errorMsg, "Error message should be displayed");
        Assert.assertTrue(errorMsg.contains("Invalid") || errorMsg.contains("incorrect"),
            "Error message should indicate invalid credentials");
        
        ExtentReportManager.logPass("âœ“ Invalid password test passed");
    }
    
    /**
     * Test: Login fails with empty username
     */
    @Test(priority = 3,
          groups = {"regression", "login", "validation"},
          description = "Verify validation for empty username")
    public void testEmptyUsername() {
        ExtentReportManager.assignCategory("Regression", "Validation");
        ExtentReportManager.logInfo("Test: Empty Username");
        
        // Act
        String errorMsg = loginFlow.performInvalidLogin("", "Password123!");
        
        // Assert
        Assert.assertNotNull(errorMsg, "Validation message should be displayed");
        
        ExtentReportManager.logPass("âœ“ Empty username validation passed");
    }
    
    /**
     * Test: Login page elements are present
     */
    @Test(priority = 4,
          groups = {"smoke", "ui"},
          description = "Verify all login page elements are displayed")
    public void testLoginPageElements() {
        ExtentReportManager.assignCategory("Smoke", "UI");
        ExtentReportManager.logInfo("Test: Login Page Elements");
        
        // Arrange
        navigateToApplication();
        LoginPage loginPage = new LoginPage(driver);
        
        // Act & Assert
        Assert.assertTrue(loginPage.isPageLoaded(), "Login page should load");
        Assert.assertTrue(loginPage.verifyAllElements(), "All elements should be present");
        
        ExtentReportManager.logPass("âœ“ Login page elements test passed");
    }
    
    /**
     * Test: Remember me functionality
     */
    @Test(priority = 5,
          groups = {"regression", "login"},
          description = "Verify remember me checkbox functionality")
    public void testRememberMe() {
        ExtentReportManager.assignCategory("Regression", "Login");
        ExtentReportManager.logInfo("Test: Remember Me");
        
        navigateToApplication();

        // Test remember me

## ğŸ’¡ Pro Tips

### 1. Start Small, Build Big
- Begin with 2-3 critical pages
- Write 5-10 smoke tests
- Expand gradually

### 2. Follow Naming Conventions
```
Pages: LoginPage, HomePage, ProductPage
Flows: LoginFlow, PurchaseFlow
Tests: LoginTest, ProductTest
Methods: testValidLogin, testInvalidPassword
```

### 3. Use Descriptive Names
```java
// âŒ Bad
@Test
public void test1() { }

// âœ… Good
@Test(description = "Verify user can login with valid credentials")
public void testValidLoginWithCorrectCredentials() { }
```

### 4. Keep Tests Independent
Each test should:
- Set up its own data
- Not depend on other tests
- Clean up after itself

### 5. Use Groups Effectively
```java
@Test(groups = {"smoke", "login"})          // Critical tests
@Test(groups = {"regression", "cart"})      // Full coverage
@Test(groups = {"negative", "validation"})  // Error scenarios
```

### 6. Log Everything
```java
ExtentReportManager.logInfo("Starting purchase flow");
ExtentReportManager.logInfo("Added product to cart");
ExtentReportManager.logPass("Order placed successfully");
```

## ğŸš¨ Common Pitfalls to Avoid

### âŒ Don't
- Hardcode URLs, credentials, or test data
- Use `Thread.sleep()` for waits
- Create dependent tests
- Put business logic in tests
- Use unstable locators (e.g., xpath with index)

### âœ… Do
- Use ConfigReader for all configuration
- Use explicit waits
- Make tests independent
- Put business logic in flows
- Use stable locators (id, name, data-attributes)
